# CPP01

Second C++ module. Covers memory allocation (`new`/`delete`), references, and pointers to members.

## Overview

- **ex00**: Zombie — heap vs stack allocation
- **ex01**: zombieHorde — array of Zombies
- **ex02**: BraiiiiiiinnnzzzZ — brain announcement
- **ex03**: Weapon — references (HumanA) vs pointers (HumanB)
- **ex04**: Sed — file find-and-replace (C++ style)
- **ex05/06**: Harl — switch/complain levels

## Setup

```bash
cd ex00 && make && ./zombie
cd ex04 && make && ./sed file s1 s2
cd ex05 && make && ./harl DEBUG
```
