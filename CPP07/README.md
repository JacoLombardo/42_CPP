# CPP07

Eighth C++ module. Templates (function and class).

## Overview

- **ex00**: whatever — function template (min, max, swap)
- **ex01**: iter — function template for iterating
- **ex02**: Array — class template with bounds checking

## Setup

```bash
cd ex00 && make && ./whatever
cd ex01 && make && ./iter
cd ex02 && make && ./array
```
