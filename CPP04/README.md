# CPP04

Fifth C++ module. Polymorphism, abstract classes, and interfaces.

## Overview

- **ex00**: Animal, Dog, Cat — virtual destructors
- **ex01**: Brain — deep copy
- **ex02**: AAnimal — pure virtual (abstract class)
- **ex03**: Materia — interfaces (IMateriaSource, ICharacter), Ice, Cure, Character

## Setup

```bash
cd ex00 && make && ./animal
cd ex03 && make && ./materia
```
