# CPP06

Seventh C++ module. C++ casts and serialization.

## Overview

- **ex00**: ScalarConverter — converts char/int/float/double literals
- **ex01**: Serializer — reinterpret_cast, data serialization
- **ex02**: Base, identify — dynamic_cast, runtime type identification

## Setup

```bash
cd ex00 && make && ./convert "42"
cd ex01 && make && ./serializer
cd ex02 && make && ./identify
```
