# CPP05

Sixth C++ module. Repetition and exceptions (Bureaucrat, Form).

## Overview

- **ex00**: Bureaucrat — grades, exceptions for out-of-range
- **ex01**: Form — Bureaucrat signs/executes forms
- **ex02**: AForm — abstract base, ShrubberyCreationForm, RobotomyRequestForm, PresidentialPardonForm
- **ex03**: Intern — creates forms by name

## Setup

```bash
cd ex00 && make && ./bureaucrat
cd ex03 && make && ./intern
```
