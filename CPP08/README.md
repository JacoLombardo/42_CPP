# CPP08

Ninth C++ module. STL algorithms and containers.

## Overview

- **ex00**: easyfind — template that finds a value in a container
- **ex01**: Span — store N ints, add numbers, shortest/longest span
- **ex02**: MutantStack — stack adapter with iterators

## Setup

```bash
cd ex00 && make && ./easyfind
cd ex01 && make && ./span
cd ex02 && make && ./mutant
```
