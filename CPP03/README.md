# CPP03

Fourth C++ module. Inheritance, virtual functions, and the Diamond Problem.

## Overview

- **ex00**: ClapTrap — base class
- **ex01**: ScavTrap — inherits ClapTrap
- **ex02**: FragTrap — multiple inheritance (ClapTrap, ScavTrap, FragTrap)
- **ex03**: DiamondTrap — virtual inheritance (Diamond Problem)

## Setup

```bash
cd ex00 && make && ./claptrap
cd ex03 && make && ./diamond
```
