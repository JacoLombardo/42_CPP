# CPP09

Tenth C++ module. Practical mini-projects.

## Overview

- **ex00**: BitcoinExchange — parse CSV, compute value from date
- **ex01**: RPN — Reverse Polish Notation calculator
- **ex02**: PmergeMe — Ford–Johnson (merge-insertion) sort for positive integers

## Setup

```bash
cd ex00 && make && ./btc input.txt
cd ex01 && make && ./RPN "8 9 * 9 - 9 - 9 - 4 - 1 +"
cd ex02 && make && ./PmergeMe "3 5 9 7 4"
```
