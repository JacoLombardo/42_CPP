# CPP00

First C++ module at 42. Introduces namespaces, classes, member functions, and basic I/O streams.

## Overview

- **ex00**: megaphone — uppercase output
- **ex01**: PhoneBook — contact management with classes
- **ex02**: Account — timestamped banking logs (19920104_091532.log)

## Setup

```bash
cd ex00 && make && ./megaphone "hello"
cd ex01 && make && ./phonebook
cd ex02 && make && ./account
```
